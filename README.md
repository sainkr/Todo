# Challenge 
## 개요
- 작업 기간 : 2020.11 ~ 2020.12
- 개발 인원 : 1명
- FrontEnd 개발 환경 : Kotlin, Android
- BackEnd 개발 환경 : JSP, Java, MySQL, Apacheomcat, eclipse

- 소개
+ 오늘 할일, 달력 확인, 친구들과 챌린지를 할 수 있는 어플리케이션

## 기능
+ 회원가입
+ 로그인 / 로그아웃
+ Todo 관리
  + Todo 추가 / 수정 / 삭제
  + 오늘 날씨 설정
+ 달력 제공
  + 날짜 클릭시 해당 Todo 확인 가능
+ 친구 관리
  + 친구 추가 / 삭제
+ 친구와 함께 Challenge
  + 챌린지 추가 / 수정 / 삭제
  + 챌린지원 추가 / 수정
  + 챌린지 내용 추가 / 수정 / 삭제
  + 챌린지원 챌린지 달성 여부 확인 가능
